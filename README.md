# Express-RESTAPI

REST (Representational State Transfer) API with Express.js

## Installation

Clone the repository:
    ```
    git clone https://github.com/pxiaccount/Django-React-CRUD.git
    ```
    
### Express
1. Change directory in to client:

    ```bash
    cd Express-RESTAPI
    ```

2. Install packages:

    ```bash
    npm i
    ```

3. Run the development server:

    ```bash
    npm run dev
    ```
